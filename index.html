Online TOML to JSON converter Online toml2json converter toml2json converter toml2json

Convert TOML to JSON or JSON to TOMl right here on this page* (toml2json, json2toml).

TOML (Tom's Obvious, Minimal Language) aims to be a minimal configuration file format that's easy to read due to obvious semantics. TOML is designed to map unambiguously to a hash table. TOML should be easy to parse into data structures in a wide variety of languages.

* The JSON to TOML conversion is done server-side for technical reasons, however your requests are not saved on the server.
